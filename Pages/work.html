Meet App

Description: In the "Meet App" project, I successfully built a serverless, progressive web application (PWA) using React and the test-driven development (TDD) approach. The application capitalizes on the Google Calendar API to fetch and display upcoming events. Through the integration of serverless technology and the features of PWAs, this project underscores my capacity to create web applications that are efficient, scalable, and user-centric.

Objective: Craft a serverless, PWA with React, employing the TDD methodology. The application should efficiently use the Google Calendar API to present upcoming events and offer users the ability to filter these events by city. Additionally, integrate data visualization to enhance the user experience.

Technical Highlights:

  • Developed using the React framework, ensuring a component-based architecture.
  • Integrated the Google Calendar API along with OAuth2 for user authentication.
  • Utilized AWS Lambda for serverless functions, providing efficient scalability and eliminating backend maintenance.
  • Achieved deployment on GitHub Pages, making the app accessible to a broad audience.
  • Designed the app to be responsive across various devices, supporting screen widths from 320px to 1920px.
  • Employed TDD to ensure high code quality, robustness, and functionality.
  • Leveraged a service worker to enable offline functionality, enhancing user access in varied network conditions.
  • Implemented data visualization using charts to provide insights on event distribution and genre popularity.
  • Ensured the app passes Lighthouse's PWA checklist, validating its quality as a PWA.
  • Adopted an Object-Oriented Programming (OOP) approach for the app's alert system.
  • Ensured a high test coverage rate of >= 90%, emphasizing the reliability and robustness of the application.
  • Integrated continuous delivery and performance monitoring to maintain app performance and quality.
